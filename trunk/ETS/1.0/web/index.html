Overview

This test suite is based on the following OGC specifications:

  • OpenGIS Web Services Common Specification [OGC 05-008c1]
  • OWS Common Implementation Specification [OGC 06-121r9]
  • OpenGIS® Implementation Standard for Geographic information - Simple feature access - Part 1: Common architecture [OGC 06-103r4]
  • Earth Observation Metadata profile of Observations & Measurements [OGC 10-157r4]
  • OpenSearch Earth Observation product Extension [OGC 10-026r2]
  • OpenSearch GeoSpatial and Temporal Extensions [OGC 10-032r6]
  • SWE Common Data Model Encoding Standard 2.0.0 [OGC 08-094]
  • Sensor Planning Service Implementation Standard 2.0 [OGC 09-000]
  • Sensor Planning Service Interface Standard 2.0 Earth Observation Satellite Tasking Extension [OGC 10-135]

The conformance test cover basic search capabilities.

What is tested

  • Correctness of the OpenSearch Description Document(s)
  • Support of  ATOM format
  • Support of search parameters in geographical and temporal format
  • Support of search parameters in string and numerical format
  • Correct format of the response (i.e. GetFeasibilityResponse returned)

What is not tested

  • The correctness of the results (i.e. matching of the results with the search parameters) is only tested for geographical, temporal and a few additional parameters (sensor type, platform, instrument). For the remaining parameters, the suite only checks that a well formed response is returned.
